# JavaScript setTimeout Example

This is a small JavaScript project showing how the `setTimeout()` function can execute code after a certain delay. I created it as part of my JS learning exercises.

## What the project does

- Demonstrates how to delay the execution of a function
- Simple example showing a message after a few seconds

## Technologies

- HTML
- JavaScript

## How to run

1. Open `index.html` in your browser
2. Observe how the `setTimeout` function works
